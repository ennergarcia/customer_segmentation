Clusterização de dados: segmentação de clientes

Este estudo de caso de ciência de dados explora uma experiência comum de marketing digital em análises para e-commerce. O objetivo deste projeto é criar clusters para segmentar os clientes da base, buscando uma maior objetividade nas ofertas e, por consequência, uma maior taxa de conversão.
